Important Return Policy Information

What Can Be Returned

  • • Products in original condition
  • • Items with all tags attached
  • • Products in original packaging
  • • Defective or damaged items

What Cannot Be Returned

  • • Intimate apparel & swimwear
  • • Personalized products
  • • Perishable goods
  • • Software & digital products

How to Return an Item

1

Initiate Return

Log into your ShopHub account, go to "My Orders" and select the item you want to return. Click "Return Item" and select your reason for return.

Available return reasons: Wrong item, Defective, Size issue, Changed mind, Better price available

2

Print Return Label

Once your return request is approved, you'll receive a return shipping label via email. Print this label and attach it securely to your package.

Note: For defective products, return shipping is free. For other returns, shipping costs may apply.

3

Pack & Ship

Pack the item in its original packaging with all accessories and tags. Include the original invoice and drop off the package at any authorized shipping location.

Packaging tips: Use the original box if possible, include all manuals and accessories, ensure the item is securely packed.

4

Receive Refund

Once we receive and inspect your return, we'll process your refund within 5-7 business days. You'll receive a confirmation email when the refund is processed.

Refund methods: Refunds are issued to the original payment method. Processing times vary by payment provider.

Exchange Policy

Product Exchanges

We offer easy exchanges for size, color, or style changes. Exchanges are processed within 2-3 business days after we receive your return.

  • Free exchange shipping
  • Same product, different variant
  • Priority processing

How to Request an Exchange

  1. 1 Initiate return for the original item
  2. 2 Select "Exchange" as return reason
  3. 3 Choose your preferred variant
  4. 4 We'll ship the new item once return is received
